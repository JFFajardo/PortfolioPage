Dog's App

Johan Dog's APP es una Single Page Application desarrollada para presentar como proyecto individual del bootcamp Soy Henry en el programa Full Stack. Consume datos de la api “thedogapi” y los presenta en un home de 8 dogs por página. Estos se pueden filtrar por temperamentos, dogs de la api o dogs creados por los usuarios; también se pueden ordenar por alfabéticamente de forma ascendente/ descendente o por su peso mínimo o máximo. Cuenta con una barra de búsqueda por nombre y un formulario que permite la creación de una nueva raza. Al posicionarse sobre cada carta y escogerse determinado dog, se podrán observar sus detalles. Cuenta con páginas de landing, error y about.


Tecnologías empleadas: -Lenguajes: JavaScript, SQL. -Base de datos: PostgreSQL, Sequelize. -Back-End: NodeJS, ExpressJS. -Front-End: React, Redux, CSS modules.
