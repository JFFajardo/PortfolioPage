HenryMoon

Rick and Morty APP es una Single Page Application desarrollada como proyecto integrador del bootcamp Soy Henry en el programa FullStack. La pagina cuenta con un login para que el usuario ingrese con su email y contraseña. Consume datos de la api rickandmortyapi y los presenta en un home cada vez que el usuario introduzca un id en la barra de búsqueda o de click en el botón de personaje random. Cada personaje se puede agregar a favoritos y asi mismo se pueden filtrar por géneros; también se pueden ordenar alfabéticamente de forma ascendente/ descendente. Al posicionarse sobre cada carta y escogerse determinado personaje, se podrán observar sus detalles. Cuenta con paginas de landing, error y about.


Tecnologías empleadas: -Lenguajes: JavaScript, SQL. -Base de datos: PostgreSQL, Sequelize. -Back-End: NodeJS, ExpressJS. -Front-End: React, Redux, CSS modules.
